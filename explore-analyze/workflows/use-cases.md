# Use cases [workflows-use-cases]

Browse these sections to see what you can accomplish with workflows. Each use case describes the patterns, triggers, and steps most relevant to it, with pointers into the reference documentation for each building block.

- [Security](/explore-analyze/workflows/use-cases/security.md): Automate alert response, case creation, enrichment, notification routing, and large-scale detection rule operations.
- [Observability](/explore-analyze/workflows/use-cases/observability.md): Respond to anomaly detection alerts, correlate signals across data sources, and automate scheduled data operations.
- [AI-augmented workflows](/explore-analyze/workflows/use-cases/ai-augmented-workflows.md): Combine workflows with {{agent-builder}} agents so deterministic automation and AI reasoning work together.
