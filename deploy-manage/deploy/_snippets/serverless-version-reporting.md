{{serverless-short}} is versionless and continuously updated. The [`GET /` root API](https://www.elastic.co/docs/api/doc/elasticsearch-serverless/group/endpoint-info) always reports the next target {{es}} release version at the time of the request. This value exists for client compatibility only and does not indicate which features are available in your project. Use `build_flavor: serverless` to confirm you are working with a Serverless project rather than relying on the version number.
